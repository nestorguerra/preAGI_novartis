## Publicación en GitHub Pages

1. Sube todos estos archivos a un repositorio de GitHub
2. Ve a **Settings → Pages**
3. En *Source*, selecciona `main` branch y carpeta `/ (root)`
4. Guarda. En unos minutos tendrás la URL: `https://TU-USUARIO.github.io/NOMBRE-REPO/`
